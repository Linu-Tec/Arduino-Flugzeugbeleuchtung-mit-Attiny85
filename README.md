## Benötigte Bauteile:
- 4x Widerstand 270Ohm
- 5x Widerstand 150Ohm
- 1x Widerstand 100Ohm
- 1x LED Grün 3mm
- 13x LED Weiß 3mm
- 3x LED Rot 3mm
- 1x ATtiny85
- 1x IC-Sockel, 8-polig



## Anschlüsse

- ATtiny85	LED

- 0	Strobe (Weiß)

- 1	Beacon (Rot)

- 2	taxi (Weiß)

- 3	TakeoffLanding (Weiß)

- 4	Runwayturnoff (Weiß)
